-------------------------------
Voorwoord
-------------------------------
Voordat u ons project gaat bekijken willen wij graag een klein woordje meedelen. U moet verstaan dat wij een zeer druk semester
achter de rug hebben. Zo hebben wij ons vak Projecten waarbij wij een back-end + front-end (Android en Web) moeten maken. Dit
project neemt enorm veel tijd in beslag waarbij elk van ons minstens 30 uur per week heeft ingestoken vanaf. Daarnaast hebben
wij nog het project van webapps waar ook veel tijd is ingekropen. Ook hadden wekelijks taken voor internationale communicatie en
moesten wij wekelijks oefeningen voor android maken. Tot slot kwamen hier taken bij voor het vak iTalent en Topics. Dit was 
voor ons dus een semester waarbij wij zeer veel moesten werken en dit heeft dus ook ons windows project beinvloed. Wij zijn elke
les aanwezig geweest, enkel de 2 laatste lessen niet omwille van het feit dat wij simpelweg te veel werk hadden. Wij hebben toch
nog ons best gedaan om de app zoveel mogelijk functionaliteit te geven en goede code te schrijven. Wij hopen dus dat u hiermee
probeer rekening te houden bij de evaluatie.

Over wat beschikt onze app nu precies?
- SQL Server Databank
- Volledig werkende backend (waaronder authenticatie)
- MVVM principes toegepast
- Een front-end app waar vooral functionaliteit werkt, de UI is basic

De app kan op enkele momenten helaas nog crashen, gelieve daarom ook onze instructies na te lezen.
